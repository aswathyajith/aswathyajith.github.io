Counting in Sight, Word, and Thought

A Study of Count Alignment in Multi-modal Text-Vision Models

October 9, 2025

Abstract

Multi-modal text-vision models have shown remarkable capabilities in generating and editing images with text prompts. However, their ability to process and generate numerical data is still under-explored. This post aims to explore the counting abilities of multi-modal text-vision models.

Introduction

In this post, I will do a deep dive into the generative and counting abilities of multi-modal text-vision models.

Experimental Setup

We examine the following aspects to understand the cognition of numerical values in multi-modal text-vision models:

  1. Counting: We prompt models to count the number of objects in a synthetically created image.
  2. Generation: We prompt models to generate an image with a specified number of objects.
  3. Consistency: We prompt models to generate the same number of objects in an image.
  4. Probing: We train linear probes on model activations across different layers and test their ability to predict the number of objects in an image.
